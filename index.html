SASS - Preprocesador CSS

Documentación Instalación
  1. Instalar sass mediante linea de comando en terminal de estacion de trabajo: npm install -g sass
  2. Crear archivo main.sass en la carpeta de archivos CSS
  3. Vincular archivo html el archivo main.css para los estilos, este será el archivo compilado.
  4. Desde la terminal posicionarse en la carpeta de estilos css del proyecto.
  5. Desde la carpeta css, ejecutar siguiente comando para compilar: sass --watch main.sass:main.css
  6. Se sugiere registrar variables para aquellas propiedades que puedan ser utilizadas varias veces en el codigo: ejemplo, color, font-family, font-size,
  7. Verificar el espacio despues de los dos puntos.
  8. Anidación de selectores mediante identación
  9. Importar archivos
    • Crear archivos que comiencen nombre con _nombre.sass
    • en el archivo main utilizar: @use: nombrearchivo
  10. Se recomienda al final que el archivo main.sass sea solo de @use
